# Datasets Source* This dataset was from the UCI ML Repository: https://archive.ics.uci.edu/ml/datasets/ILPD+(Indian+Liver+Patient+Dataset## Dataset Information* This training dataset contains 467 records, including 333 liver patient records and 134 non liver patient records. The data set was collected from north east of Andhra Pradesh, India. Label field is a class label used to divide into groups(liver patient or not). Any patient whose age exceeded 89 is listed as being of age "90".## Attribute Information:1. Age: Age of the patient (年齡)2. Gender: Gender of the patient (性別)3. TB: Total Bilirubin (總膽紅素)4. DB: Direct Bilirubin (直接型膽紅素/結合型膽紅素)5. Alkphos: Alkaline Phosphotase (鹼性磷酸酶)6. Sgpt: Alamine Aminotransferase (麩胺酸丙酮酸轉氨基酶/GPT)7. Sgot: Aspartate Aminotransferase (麩胺酸苯醋酸轉氨基酶/GOT)8. TP: Total Protiens (總蛋白)9. ALB: Albumin (白蛋白)10. A/G Ratio: Albumin and Globulin Ratio (白蛋白/球蛋白比值)11. Label: used to split the data into two sets## Additional information[如何解讀肝功能檢驗報告](https://www.jah.org.tw/form/index-1.asp?m=3&m1=8&m2=366&gp=361&id=522)
